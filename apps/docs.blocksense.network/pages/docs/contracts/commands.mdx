# Development Setup and Commands

### Display Available Commands

See a list of all available Hardhat commands and their descriptions.

```bash copy
yarn hardhat help
```

### Clean All Untracked Files

This command removes all untracked files and directories. It's useful for cleaning up your workspace.

```bash copy
yarn clean
```

### Compile Contracts

[Compile](https://github.com/blocksense-network/blocksense/blob/a7dc2c4976f6c8fee6a913616c29bf948d489bbc/libs/contracts/package.json#L12) smart contracts to ensure they are up-to-date and ready for deployment or testing.

```bash copy filename="package.json"
yarn build
```

### Run Tests

[Execute](https://github.com/blocksense-network/blocksense/blob/a7dc2c4976f6c8fee6a913616c29bf948d489bbc/libs/contracts/package.json#L14) the test suite to verify that the contracts behave as expected.

```bash copy filename="package.json"
yarn test
```

### Run Tests with Forking Enabled Only for Tests Marked with @fork in the Description

[Run](https://github.com/blocksense-network/blocksense/blob/a7dc2c4976f6c8fee6a913616c29bf948d489bbc/libs/contracts/package.json#L15) tests with forking enabled for specific tests marked with @fork.

```bash copy filename="package.json"
yarn test:fork
```

### Run Tests and Generate Coverage Report

[Generate](https://github.com/blocksense-network/blocksense/blob/a7dc2c4976f6c8fee6a913616c29bf948d489bbc/libs/contracts/package.json#L16) a coverage report.

```bash copy filename="package.json"
yarn coverage
```

### Display Size of Contracts

Display the [size](https://github.com/blocksense-network/blocksense/blob/a7dc2c4976f6c8fee6a913616c29bf948d489bbc/libs/contracts/package.json#L18) of the compiled contracts to monitor their deployment costs.

```bash copy filename="package.json"
yarn size
```

### Run Local Nodes in Separate Shells

Start a local node and [deploy](https://github.com/blocksense-network/blocksense/blob/a7dc2c4976f6c8fee6a913616c29bf948d489bbc/libs/contracts/package.json#L17) contracts to it. This is useful for local testing and development.

```bash copy filename="package.json"
yarn hardhat node
yarn deploy:local ./scripts/<script_name>.ts
```

### Fork Mainet

Fork the mainnet to test scripts or contracts in a controlled local environment.

```bash copy
FORKING=true yarn hardhat run ./scripts/{script-name}.ts
```

### Run Tests with Opcodes & Gas Tracing

Run tests with opcode and gas usage tracing enabled to analyze the performance and efficiency of your contracts.

```bash copy
TRACE_TX=true yarn test
```
