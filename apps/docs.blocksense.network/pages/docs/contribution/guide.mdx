# How to Contribute

## Guidelines for Contributors

1. Fork the [GitHub](/coming-soon) repository.
2. Create a new branch for your feature or bugfix:

```bash copy
git checkout -b feature/my-new-feature
```
3. Make your changes and commit them with a descriptive message:

```bash copy
git commit -am 'Add new feature'
```
4. Push your changes to your fork

```bash copy
git push origin feature/my-new-feature
```

## Code Style

Ensure your code follows the project's style guidelines. Run the linter to check for issues

```bash copy
yarn lint
```

## Running Tests

Before submitting your pull request, run the tests to make sure that everything is working correctly

```bash copy
yarn test
```
## Document your work
If your contribution affects the public API of the project or requires users to take any action, please update the documentation accordingly

### Questions?
If you have any questions or need assistance, feel free to open an issue or contact us at hi@blocksense.network.
